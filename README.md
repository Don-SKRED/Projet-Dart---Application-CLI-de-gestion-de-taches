# Task Management CLI

Une application Dart en ligne de commande pour gérer vos tâches quotidiennes.

## Fonctionnalités

- Ajouter une nouvelle tâche
- Lister toutes les tâches avec tri (par priorité, par date)
- Marquer une tâche comme terminée
- Supprimer une tâche
- Persistence des données en JSON

## Installation

### 1. Cloner le projet

\`\`\`bash
git clone https://github.com/Don-SKRED/Projet-Dart---Application-CLI-de-gestion-de-taches.git
cd Projet-Dart---Application-CLI-de-gestion-de-taches
\`\`\`

### 2. Récupérer les dépendances

\`\`\`bash
dart pub get
\`\`\`

## Utilisation

### Lancer l'application

\`\`\`bash
dart run
\`\`\`
